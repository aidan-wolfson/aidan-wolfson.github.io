Software Engineering

Projects & Playable Demos

'I Forgot Something!' was my submission to Adventure Jam 2023 (and first jam submission EVER). It's a small adventure game about a young boy journeying through his childhood backyard for the last time. I learned so much while developing this project, from adventure game puzzle design using dependency charts (thank you for the lesson, Ron Gilbert's Blog) to realistic project scoping for jams! Implementing the dialogue system was a challenge, but in my next project I'd like to expand on the foundation I laid in this game to achieve a more modular and efficient system for multiple interactions with NPCs. The system I'm most proud of would have to be the achievement system, which very simply plays a little victory jingle and displays achievement text whenever the player achieves one of the few milestones in the game. Even though it's a small and simple addition, I think it adds a lot to the game feel and really elevates the experience.

Overall, I learned so much from participating in this jam and finishing this small project. I'm very excited to take all that I've learned and apply it to my next project!!
